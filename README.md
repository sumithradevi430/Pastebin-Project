# Pastebin Application

A Spring Boot based Pastebin web application that allows users to create, share,
and auto-expire text pastes.

## Features
- Create and share text pastes
- Expiration based on time and views
- Spring Boot + Thymeleaf UI
- MySQL / H2 support

## Tech Stack
- Java
- Spring Boot
- Thymeleaf
- MySQL
- Maven
